항공사 FAQ RAG 시스템

이 프로젝트는 항공권 서비스에서 자주 묻는 질문(FAQ)을 항공사별 JSON 파일로 분리해 저장하고,
Python을 이용해 사용자가 입력한 항공사명에 따라 해당 FAQ 데이터를 불러와 RAG(Retrieval-Augmented Generation) 에 활용할 수 있도록 설계된 시스템입니다.

현재 지원하는 항공사는 총 5곳입니다.

진에어

에어부산

티웨이항공

제주항공

에어프레미아

각 항공사별 FAQ는 개별 .json 파일로 저장되며, RAG 시스템에서 벡터 DB 검색 기반으로 활용할 수 있습니다.
